# Trust Leaf Go-To-Market Kit

Este paquete resume Trust Leaf para salir a conversar con grants, capital,
pilotos y aliados estrategicos.

## Entregables

- [One-pager ejecutivo](./one-pager.md): version corta para enviar por mail,
  formulario de grant o primer contacto.
- [Pitch deck breve](./pitch-deck.md): estructura slide por slide para una
  presentacion de capital/grants.
- [Plan de negocio](./business-plan.md): modelo comercial, flujo de dinero,
  costos, hitos y uso de fondos.
- [Modelo financiero](./financial-model.md): escenarios de ronda, presupuesto
  de 12 meses, posibles ingresos y fees de red.
- [Guion de video](./video-script-60s.md): version de 1 minuto para presentar
  el proyecto de forma humana y directa.

## Posicionamiento

Trust Leaf no es un marketplace generico de cannabis. Es una red de salud y
trazabilidad para cannabis medicinal donde cada actor opera desde su propio
punto de vista:

- paciente: controla identidad, historial clinico privado, recetas y accesos;
- medico: revisa evidencia autorizada y emite recetas verificables;
- dispensario: valida receta, cupo y lote sin ver historia clinica completa;
- admin: aprueba actores validos y mantiene la red confiable.

La tesis central: informacion valida, privada y verificable.

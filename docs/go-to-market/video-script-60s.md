# Guion de video: 60 segundos

## Version principal

Soy [tu nombre], fundador de Trust Leaf, y tambien soy paciente cannabico.

Estamos construyendo una red privada y verificable para cannabis medicinal.
Hoy el acceso sigue fragmentado: recetas dificiles de validar, pacientes que
repiten su historial clinico, dispensarios que no saben con claridad que pueden
entregar y productos con poca trazabilidad por lote.

Trust Leaf separa lo que ve cada actor. El paciente controla su informacion,
examenes y recetas. El medico valida evidencia y emite una receta verificable.
El dispensario confirma solo lo necesario para entregar y registra cada retiro
parcial. El admin aprueba medicos y dispensarios validos.

Usamos Stellar para pruebas verificables y agentes 402 para validar informacion
sensible sin exponer documentos completos. La historia clinica queda privada;
la red solo prueba estado, permiso y trazabilidad.

Ya tenemos un MVP en produccion con portales separados para paciente, medico,
dispensario y admin. El siguiente paso es convertirlo en pilotos reales en
LATAM, con smart contracts robustos, base de datos segura, passkeys y aliados
del ecosistema cannabis medicinal.

Buscamos capital, grants y partners para construir la infraestructura de
confianza que este mercado necesita.

## Version mas emocional

Soy [tu nombre], fundador de Trust Leaf, y este proyecto nace desde una realidad
personal: soy paciente cannabico.

Cuando necesitas cannabis medicinal, no basta con que el producto exista.
Necesitas acceso seguro, medicos confiables, recetas validas, dispensarios que
entiendan que pueden entregar y privacidad para no exponer tu historia clinica
cada vez que pides ayuda.

Trust Leaf es una red privada y verificable para resolver eso. El paciente
controla sus datos. El medico valida y emite recetas verificables. El
dispensario entrega medicina trazable sin ver informacion clinica completa. Y
el admin mantiene la red de actores autorizados.

Con Stellar registramos pruebas verificables. Con agentes 402 validamos datos
sensibles sin revelar documentos completos.

Ya tenemos un MVP funcionando en produccion. Ahora queremos llevarlo a pilotos
reales en LATAM con medicos, dispensarios y marcas del ecosistema.

Buscamos capital, grants y aliados para construir la infraestructura de
confianza del cannabis medicinal.
